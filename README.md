**🌟 Project Overview**

Resume Role Analyzer is a web-based application that automatically analyzes resumes (PDF/DOCX) and matches candidate skills to specific job roles using Deep Learning (SentenceTransformers) and cosine similarity.

It helps recruiters shortlist candidates efficiently and provides actionable insights for candidates to improve their resumes.
⚙️ Features

Upload resumes in PDF or DOCX format.

Extract skills and experience from resumes.

Role-based matching with semantic understanding.

Highlight matching and missing skills.

Generate downloadable PDF reports.

User-friendly web interface built with Flask.
🛠️ Setup Instructions

1.Clone or download this repository:
git clone https://github.com/Soudhamini04/resume-analyzer.git
cd resume-analyzer
2.Create a virtual environment:
# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python -m venv venv
source venv/bin/activate

3.Install dependencies:
python -m pip install -r requirements.txt
